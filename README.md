# Тестовое задание - Выбор вещей

Приложение для выбора и обмена вещей, реализованное на Vue 3.

## Описание

Интерфейс состоит из 4 блоков:
- Верхний левый: выбранные вещи пользователя (до 6 штук)
- Верхний правый: выбранные вещи на обмен
- Нижний левый: доступные вещи пользователя
- Нижний правый: доступные вещи на выбор

## Механика

- Можно выбрать от 1 до 6 вещей из списка пользователя
- Количество вещей на обмен равно количеству выбранных вещей пользователя
- При уменьшении количества вещей пользователя, автоматически уменьшается количество вещей на обмен

## Установка и запуск

```bash
npm install
npm run dev
```

## Технологии

- Vue 3
- Vite
